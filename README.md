# packman-gui
Universal GUI for Node.js Dependencies Management. Compatible with npm and Yarn.

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
